Computer System Overview: 
    - Input: provide data to system ex: mouse, keyboard etc
    - output: show result data ex: monitor, printer and speaker etc
    - RAM: It is fastest than storage, low memory and it stores data temparary, 
        - If shutdown then data removed
        - it stores data that currently process and this data used by CPU
    - Storage: It is slow, large memory and it stores data Permanent
    - CPU: it fetch data from Usually RAM often through cache, decode and exicute instructions, it uses register to hold current data and addresses
    - OS: it is manage CPU Scheduling(time), memory, devices etc without this applications not handle hardware devices, it provide sevices to application


CPU Basics:
    - CPU(central processing unit) it fetch instruction/program from ram, decode it means understand the instructions then it exicutes the instructions, the below are cpu parts
    - Main parts of CPU: Here only 4 parts i am writing but there are more is there this 4 is enough for OS
        - CU(control unit): 
            - It fetch instruction from RAM
            - It decode the instructions 
            - It controls the Exicute instructions
            - It tell to other components what operations have to do in exicute instructions
        - ALU(Airthmatic Logical Unit): It is used to perform calculations
        - Register: It is fastest storage inside cpu, it holds the data and addresses of currently working instructions/program
        - cache: It stores frequently used data and instructions so the cpu dosen't need to access repeatedly from ram, and it is near to cpu and comparing to faster response cache is more faster than RAM, but it have small memory
