# 1. What is an Operating System?

### Simple Definition

An Operating System is software that acts as a bridge between applications and computer hardware.

Examples:

* Windows
* Linux
* macOS
* Android

### Why do we need an OS?

Without an OS, every program would need to directly control:

* CPU
* Memory (RAM)
* Storage devices
* Keyboard and mouse
* Network hardware

The OS manages these resources and provides a convenient interface to applications.

### Key Responsibilities

* Process management
* Memory management
* File systems
* Device management
* Security and permissions

### Learn This First

Be able to answer:

> "What problems does an OS solve?"

# 2. Kernel vs User Space

This is one of the most important OS concepts.

### User Space

Applications run here.

Examples:

* Browser
* VS Code
* Music player
* Games

User programs cannot directly access hardware because that would be unsafe.

### Kernel Space

The kernel is the core part of the OS.

It has full access to:

* CPU
* RAM
* Disk
* Hardware devices

### How They Interact

```
Application
     ↓
System Call
     ↓
Kernel
     ↓
Hardware
```

Example:

When you open a file:

```
Your Program
      ↓
open()
      ↓
Kernel
      ↓
Disk
```

The `open()` request is a **system call**.

### Important Terms

* Kernel mode (privileged)
* User mode (restricted)
* System calls
* Context switch

### Visualization

```
+------------------+
| User Space       |
| Chrome, VS Code  |
+------------------+
| System Calls     |
+------------------+
| Kernel Space     |
| Scheduler        |
| Memory Manager   |
| File System      |
+------------------+
| Hardware         |
+------------------+
```

# 3. Types of Operating Systems

You don't need deep details initially.

## Batch OS

Jobs are collected and executed in batches.

Example:

* Early mainframe systems

## Multiprogramming OS

Multiple programs stay in memory and share the CPU.

Goal:

* Improve CPU utilization

## Time-Sharing OS

CPU time is divided among users/processes.

Examples:

* Modern Linux
* Windows

## Real-Time OS (RTOS)

Must respond within strict deadlines.

Examples:

* Aircraft systems
* Medical devices
* Industrial controllers

Examples:

* FreeRTOS
* VxWorks

## Distributed OS

Multiple computers appear as one system.

## Network OS

Provides networking and resource sharing.
